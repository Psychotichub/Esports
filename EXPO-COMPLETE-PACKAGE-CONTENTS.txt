================================================================================
🚀 QUICK START
================================================================================

Frontend:
1. Extract zip file
2. cd 4psychotic-mobile
3. npm install
4. npx expo login
5. npx expo publish

Backend:
1. cd server
2. npm install
3. pnpm db:push
4. npm run build
5. npm start

YOUTUBE_API_KEY=your-youtube-api-key
YOUTUBE_CHANNEL_ID=your-channel-id
JWT_SECRET=your-secret-key
